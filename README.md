
MY BLOG WEB APP
A simple django blog app featuring post creation, 
user authentication and commenting system.

Feartures:
Create and manage posts
User authentication with Google verification
Commenting
Like/Dislike functionality

Installation
Clone this repository
Create vitual enviroment
Activate enviroment
Install requirements 
Run migrations
Create Super User
Run server

Home page to show all posts
Users interact with posts
